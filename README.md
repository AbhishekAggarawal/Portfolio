# Abhishek Aggarwal

AI Engineer · Data Scientist · Data Engineer

## Overview

This portfolio showcases AI and data engineering work built with Next.js, Tailwind CSS, and modern React. It highlights experience in data pipelines, machine learning model development, cloud analytics, and production-ready AI systems.

## Features

- AI-focused hero section with role-specific messaging
- Data engineering and analytics metrics
- Contact form for ML, analytics, and data engineering inquiries
- Animated visuals and responsive layout

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Nodemailer

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.

## Deployment

This project can be deployed to Vercel or any platform that supports Next.js App Router applications.
